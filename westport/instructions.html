this program is a simple interface to the information in freebase.com.
it can respond appropriately to several hundred million queries, about anything.

Freebase + Ubiquity from spencer cooly on Vimeo.

search

if the query is simple, it will do a keyword search.
ralph wiggum
university of baghdad
banana
chelsea clinton
if it thinks its a question but it's not, force a seach by typing:
search what is the what
or ? party of five

lists

if you search for something that is a 'type of thing', it will list (some of) them.
pets
golfers
satellites
force a list-response by typing:
list of astronauts

search filters

you can make your search smarter by specifying what the results should be.
brian politician will list politicians named brian. brian (politician) works too.
washington city
romeo and juliet movie

you can force a filtered search by typing:
winston churchill (school)
or towns named dildo

some handy freebase searches:
user spencer will find freebase users named spencer
astronomy base will just search for freebases 'bases'

website identification

if you enter an internet link, this program will be able to identify it, or what it's about.
http://en.wikipedia.org/wiki/Tunisia - (a link it gets from freebase's >uri templates)
http://www.apple.com - (a link it finds from stored weblinks)
http://spencerhold.wordpress.com/2010/01/21/how-many-people-are-in-space/ - (a page it doesn't know, but with outgoing links it does)
http://www.chass.utoronto.ca/~ikalmar/illustex/baudriterror.htm - (a page with no links that it sends to zemanta)
use this service independently here

time server

if you ask current time in namibia it will figure out it's time according to its location, utc and dst offset.
time in moose jaw
time at the cn tower
if you ask it the timezone of something that has multiple timezones, it will answer its 'middle timezone' - current time in russia

about

 this program works by using the prepositions [of, in, by, at..] in your query to squeeze it into a question that it can throw at freebase.
if there are no prepositions, it just permutates the words.

Freebase has a bunch of great apis, which made making this program pretty easy.
it has been written to fail nicely - it should, at the least, drop you off infront of a good wikipedia article.

if your query is supposed to have an 'apostrophe s', use it. don't be lazy. (it's a preposition)

in addition to the source code, all the data that this program uses is written into freebase, so editable by anyone.

If it doesn't answer a question that you think it should, add an alias to a topic, type, or property.

This is usually the problem.

Making this program, I created almost 3000 aliases for freebase properties:
[albums = 'records', sport_official = 'referee', birth_date = 'date of birth'] etc.

i am not actually a very strong programmer, and this program is, if anything, a function of the collaberative power of cool people on the internet.
It uses open-source code from Linker, Hey freebase!, Pluralizer, Sizzle, Geonames, JQuery, Propbox, Cubed, Acre
It is ad-free, stores no information, and was made to impress my colleagues.
